== Description ==
Clone Post is a WordPress plugin that allows you to easily clone or duplicate posts within your WordPress site. It provides a simple and intuitive interface to clone posts with all their associated metadata, such as categories, tags, custom fields, and more.

== Features ==
- Clone posts with a single click
- Preserve post metadata, including categories, tags, custom fields, and more
- Supports cloning of both published and draft posts
- Lightweight and easy to use

== Installation ==
1. Upload the `clone-post` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the post editor screen and click on the "Clone" button to clone a post

== Frequently Asked Questions ==
Q: How do I clone a post?
A: Simply go to the post editor screen and click on the "Clone" button. The cloned post will be created with the same content and metadata as the original post.

Q: Can I clone both published and draft posts?
A: Yes, Clone Post supports cloning of both published and draft posts.
